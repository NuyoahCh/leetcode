# LeetCode 题解

基于 Go 编写的个人刷题与题解记录。按时间划分，包含详细的思路解析与代码实现。

## 仓库结构

```
├── methodology/    # 算法方法论与总结
└── practice/       # 刷题记录
    └── 2025/
        ├── 202502/
        ├── 202503/
        └── ...
```

题解文件命名格式：`YYMMDD-题号. 题目名称.md`，如 `250204-922. 按奇偶排序数组 II.md`

## 题解内容

每篇题解包含：
- 题目链接
- 多种解法的 Go 实现
- 详细注释与思路分析
- 复杂度分析

## 快速开始

```bash
git clone https://github.com/NuyoahCh/leetcode.git
cd leetcode
go version  # 确认 Go ≥ 1.23
```
